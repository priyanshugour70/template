# docs/

Service documentation.

| Folder | Purpose |
|--------|---------|
| `swagger/` | Generated by `make swagger`. Contains `docs.go`, `swagger.json`, and `swagger.yaml`. Served by the API at `/swagger/index.html`. Do not edit by hand. |
| `samples/` | Sample payloads, import files, or curl scripts useful for API consumers and partners. |

Add hand-written design docs, runbooks, and integration guides directly under `docs/`. Anything that helps onboarding a new engineer or integrating with the API belongs here.
